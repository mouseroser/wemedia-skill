### 关键发现与修复

1. **mode=session 不可用**：Telegram channel 不支持 subagent thread binding，`mode=session` + `thread=true` 报错 "thread=true is unavailable because no channel plugin registered subagent_spawning hooks"

2. **mode=run 的 announce 回调限制**：review 作为编排中心用 mode=run 时，spawn 子 agent 后只能处理 ~3 轮 announce 就结束（标记 done），无法完成 6+ 轮的完整流水线。runTimeoutSeconds=900 无效，问题不是超时而是 run 提前结束。

3. **架构重构：main 编排模式**
