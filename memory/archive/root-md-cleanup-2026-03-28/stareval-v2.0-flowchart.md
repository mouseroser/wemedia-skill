## 核心创新 🚀

### 1. 轻量宪法 + NotebookLM 主研究双引擎
- **Step 2A**: OpenAI 宪法简报（1-2 页核心约束）
- **Step 2B**: NotebookLM 深度研究（宪法作为首个 source）
- **分离职责**: 规则驱动 + 资料驱动，避免"资料漂移"

### 2. NotebookLM 定位升级
- **项目级研究操作系统**
- **证据池**: 持续扩源，交叉验证
- **知识库**: 历史报告，技术文档
- **研究产物工厂**: 5 个标准产物

### 3. Evidence Matrix 标准化
```
| 结论 | 证据 | 来源 | 反例 | 风险 | 置信度 |
|------|------|------|------|------|--------|
```

### 4. Main 直接编排
- **去掉 review 中间层**
- **Step 4**: main → gemini
- **Step 5**: main → openai|claude
- **收益**: 效率提升 15-20%
