## 08:36 Layer 3 根因二次校正：`--session-id` 不够，需绕开 gateway lane

- 新的最小可靠修复：把 `memory-lancedb-pro/src/tools.ts` 的 Layer 3 fallback 调用从 `openclaw agent --agent notebooklm ...` 改为**直接调用** `~/.openclaw/skills/notebooklm/scripts/nlm-gateway.sh query --agent notebooklm --notebook memory-archive --query ...`，保留现有 `layer3Fallback.agent/notebook/notebookId/timeout/triggers` 配置契约不变，但彻底绕开 gateway lane 锁争用。
