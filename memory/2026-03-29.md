## 关键技术结论

### NotebookLM 正确命令（已验证）
```bash
notebooklm create "temp-{主题}-$(date +%s)"
notebooklm use <id>
notebooklm source add /tmp/file.txt
notebooklm language set zh_Hans
notebooklm generate infographic \
  --orientation square \   # 必须加，否则输出横图
  --style bento-grid \
  --language zh_Hans \
  --detail detailed \
  --wait "..."
cd /output/dir && notebooklm download infographic filename.png --force
notebooklm use <id> && notebooklm delete -y
```

### sessions_send 双向通信（已验证）
- wemedia → main：`sessions_send(sessionKey="agent:main:main", message="...")`
- main → wemedia：`sessions_send(label="wemedia-D-publish", message="...")`
- 比 announce 可靠，Step 6/7.5 必须用

### persistent session 限制
- Telegram 直聊不支持 `thread=true`，`mode=session` 无法使用
- 只能用 `mode=run` isolated session
- 通过 sessions_send 硬性回传 main 补偿 announce 不可靠问题
