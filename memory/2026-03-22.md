### NotebookLM 信息图路线探索成功
- **发现**：NotebookLM CLI 原生支持 `notebooklm generate infographic`，支持 `--orientation square`（1:1 正方形）、`--style`（bento-grid/professional 等）、`--detail`（concise/standard/detailed）、`--language zh_Hans`（简体中文）
- **优势**：全自动生成中文信息图，无需手动操作浏览器，2048x2048 原生输出，质量高
- **操作流程**：`notebooklm use <notebook>` → `notebooklm source add <txt>` → `notebooklm generate infographic --orientation square --style bento-grid --detail detailed --language zh_Hans` → `notebooklm artifact poll <id>` → `notebooklm download infographic <id>`
- **已验证效果**：A 篇信息图 2048x2048 正方形，全中文，bento-grid 风格，效果优秀

```bash
notebooklm create "标题"                    # 创建 notebook
notebooklm use <id>                        # 切换当前 notebook
```
