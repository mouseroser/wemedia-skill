## 记忆 3: write 工具常见错误
**类型**: decision
**日期**: 2026-03-08
**内容**: 调用 write 工具时容易忘记传 content 参数。根本原因是在构思长内容时注意力集中在内容本身，切换到调用工具时只想着"要写文件"，忘了"要写什么"。

**关键点**:
- write 工具必须同时传 path 和 content
- 构思内容和调用工具要分开思考
- 看到错误后要停下来重新思考，避免重复循环
