## 正文主稿

你有没有这种感觉——

让 Claude Code 跑一个任务，它开始"工作"了，然后你就只能盯着光标闪烁……10分钟过去了，它到底在干嘛？读文件？跑测试？还是卡死了？完全不知道，只能干等。

这种"AI黑盒焦虑"真的很真实，而 **Claude HUD** 就是专门解决这个问题的。

---

**Claude HUD 是什么？**

一个开源的 VSCode 扩展（支持浏览器版），给你的 Claude Code 装上可视化仪表盘 📊

实时告诉你：
- 🗂️ 它现在在读哪个文件
- ⚙️ 正在执行什么命令
- 📈 整体任务进度到哪步了
- ⏱️ 已经跑了多久

登上 GitHub 日榜 #1，今天新增 1074 star，总计 9.5K——数据说明开发者真的需要这个东西，不是我一个人有这种困扰。

---

**什么时候你最需要它？**

① **跑长任务时** — 不用无聊等待，随时看进度，知道快完成了还是刚开始

② **Agent 卡住时** — 出问题第一时间知道卡在哪一步，比翻 log 快多了

③ **团队协作时** — 做 Coding Agent 使用情况的合规审计，一目了然，不担心 AI 跑偏
